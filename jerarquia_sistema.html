Sistema de Archivos de la Jerarquía Estándar de Linux

  • /: Es la raíz del sistema de archivos y contiene todas las demás carpetas.
  • /bin: Contiene archivos binarios esenciales, como el intérprete de comandos y otros programas básicos necesarios para el arranque del sistema.
  • /boot: Contiene los archivos necesarios para el arranque del sistema, como los archivos del kernel y los archivos de configuración de GRUB.
  • /dev: Contiene los archivos de dispositivos, como los archivos de dispositivo para los discos duros, los dispositivos USB y otros dispositivos de hardware.
  • /etc: Contiene archivos de configuración del sistema y de los programas instalados.
  • /home: Contiene las carpetas personales de los usuarios.
  • /lib: Contiene las bibliotecas compartidas que utilizan los programas en el sistema.
  • /media: Contiene los puntos de montaje para los dispositivos extraíbles, como discos duros externos y dispositivos USB.
  • /mnt: Contiene los puntos de montaje para los sistemas de archivos temporales y otros sistemas de archivos adicionales.
  • /opt: Contiene los programas adicionales que no son parte del sistema operativo principal.
  • /proc: Contiene información del sistema en tiempo real, como información del hardware y del sistema operativo.
  • /root: Es la carpeta personal del usuario root.
  • /run: Contiene información de sistema en tiempo de ejecución, como archivos de PID y sockets.
  • /sbin: Contiene los programas binarios esenciales para la administración del sistema.
  • /srv: Contiene datos específicos del servicio, como archivos de configuración y datos de aplicaciones.
  • /sys: Contiene información de dispositivos y controladores.
  • /tmp: Contiene archivos temporales.
  • /usr: Contiene programas, bibliotecas, archivos de datos y documentación para aplicaciones de usuario.
  • /var: Contiene archivos que cambian frecuentemente, como registros de sistema, archivos de correo y archivos temporales.
