Ejemplo de script para usar con Cron

Para ejecutar un script con Cron, sigue los siguientes pasos:

  1. Crea el archivo del script que deseas ejecutar. Por ejemplo, script.sh.
  2. Dale permisos de ejecución al archivo del script. Puedes hacer esto con el siguiente comando: chmod +x script.sh.
  3. Abre el archivo /etc/crontab en tu editor de texto favorito y agrega la línea que ejecutará el script. Por ejemplo, si deseas ejecutar el script cada día a las 2 a.m., agrega la siguiente línea: 0 2 * * * /path/to/script.sh. Recuerda reemplazar /path/to/ con la ruta real del archivo.
  4. Guarda el archivo /etc/crontab y ciérralo.

Una vez que hayas configurado Cron para ejecutar el script, asegúrate de que el script haga lo que necesitas. Aquí hay un ejemplo de un script que hace una copia de seguridad de un directorio:

Ejemplo de script de copia de seguridad

Este script realiza una copia de seguridad de un directorio y lo comprime en un archivo tar.

#!/bin/bash
# Directorio que se desea respaldar
DIR="/path/to/backup"
Nombre del archivo de respaldo
FILE="backup_$(date +%Y%m%d).tar.gz"

Comprimir el directorio
tar -czf $FILE $DIR

Para utilizar este script con Cron, asegúrate de darle permisos de ejecución como se describe en el paso 2 anterior y configura Cron para que lo ejecute según sea necesario como se describe en los pasos 3 y 4.
