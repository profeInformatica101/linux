Tabla de comandos de Linux

Comando Descripción Ejemplo
ls Lista el contenido de un directorio. ls -l
cd Cambia el directorio actual. cd /home/usuario
mkdir Crea un nuevo directorio. mkdir nuevo_directorio
rmdir Elimina un directorio vacío. rmdir directorio_vacio
rm Elimina archivos y directorios. rm -r directorio_con_archivos
touch Crea un nuevo archivo vacío o actualiza la marca de tiempo de un archivo existente. touch archivo.txt
cp Copia archivos y directorios. cp archivo.txt copia_archivo.txt
mv Mueve o renombra archivos y directorios. mv archivo.txt carpeta/
cat Muestra el contenido de un archivo o concatena varios archivos. cat archivo.txt
chmod Modifica los permisos de archivos y directorios. chmod 755 archivo.txt
chown Cambia el propietario y el grupo de archivos y directorios. chown usuario:grupo archivo.txt
find Busca archivos y directorios en base a diferentes criterios. find /home -name "*.txt"
grep Busca patrones de texto en archivos. grep "palabra" archivo.txt
df Muestra el espacio libre y utilizado en el sistema de archivos. df -h
du Estima el espacio utilizado por archivos y directorios. du -sh /home/usuario
ps Muestra información sobre los procesos en ejecución. ps aux
top Muestra información en tiempo real sobre los procesos en ejecución y el uso de recursos del sistema. top
kill Envía una señal a un proceso para detenerlo. kill -9 12345
uname Muestra información sobre el sistema operativo y el hardware. uname -a
ping Verifica la conectividad de red con otro host. ping 8.8.8.8
ifconfig Muestra y configura las interfaces de red. ifconfig
netstat Muestra información sobre las conexiones de red, tablas de enrutamiento y estadísticas de la red. netstat -tuln
wget Descarga archivos de Internet a través de HTTP, HTTPS o FTP. wget https://ejemplo.com/archivo.zip
curl Transfiere datos desde o hacia un servidor, utilizando uno de los protocolos compatibles (HTTP, HTTPS, FTP, etc.). curl https://ejemplo.com/archivo.txt -o archivo_descargado.txt
