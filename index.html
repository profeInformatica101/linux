Ubuntu CLI Cheat Sheet 2025

Resumen operativo en acordeones de Bootstrap, con iconos e imágenes.

uname -a
— Muestra toda la información del sistema.
hostnamectl
— Muestra el nombre del host y detalles relacionados.
lscpu
— Lista la información de la arquitectura de la CPU.
timedatectl status
— Muestra la hora del sistema.

top
— Procesos en tiempo real.
htop
— Visor interactivo (requiere instalación).
df -h
— Uso de disco en formato legible.
free -m
— Memoria libre y usada (MB).
kill <process id>
— Termina un proceso.

<comando> &
— Ejecuta en segundo plano.
jobs
— Muestra comandos en background.
fg <número>
— Trae al primer plano.

sudo systemctl start <service>
— Inicia un servicio.
sudo systemctl stop <service>
— Detiene un servicio.
sudo systemctl status <service>
— Estado de un servicio.
sudo systemctl reload <service>
— Recarga la configuración sin interrumpir.
journalctl -f
— Sigue el journal en tiempo real.
journalctl -u <unit_name>
— Logs de una unidad concreta.

crontab -e
— Edita los cron jobs del usuario.
crontab -l
— Lista los cron jobs del usuario.

ls
— Lista archivos y directorios.
touch <archivo>
— Crea o actualiza un archivo.
cp <origen> <destino>
— Copia archivos.
mv <origen> <destino>
— Mueve o renombra.
rm <archivo>
— Elimina un archivo.

pwd
— Ruta del directorio actual.
cd <directorio>
— Cambia de directorio.
mkdir <nombre>
— Crea un directorio.

chmod [quién][+/-][permisos] <archivo>
— Cambia permisos.
chmod u+x <archivo>
— Hace ejecutable para el dueño.
chown [usuario]:[grupo] <archivo>
— Cambia propietario y grupo.

find [directorio] -name <patrón>
— Busca archivos y directorios.
grep <patrón> <archivo>
— Busca un patrón en archivos.

tar -czvf <nombre.tar.gz> [archivos]
— Comprime en tar.gz.
tar -xvf <nombre.tar.[gz|bz|xz]> [destino]
— Extrae un tar comprimido.

nano <archivo>
— Editor Nano.
cat <archivo>
— Muestra contenido.
less <archivo>
— Paginado.
head <archivo>
— Primeras líneas.
tail <archivo>
— Últimas líneas.
awk '{print}' <archivo>
— Imprime cada línea.

stat <archivo>
— Muestra metadatos (modo, dueño, tamaño, tiempos atime/mtime/ctime).
stat -c '%U:%G %a %s %y %n' <archivo>
— Formato personalizado: dueño:grupo, permisos (octal), bytes, mtime, nombre.
stat -f <ruta>
— Muestra información del sistema de archivos de la ruta.

sudo apt install <paquete>
— Instala un paquete.
sudo apt install -f --reinstall <paquete>
— Reinstala paquete roto.
apt search <paquete>
— Busca paquetes APT.
apt-cache policy <paquete>
— Versiones disponibles.
sudo apt update
— Actualiza índices.
sudo apt upgrade
— Actualiza paquetes.
sudo apt remove <paquete>
— Elimina un paquete.
sudo apt purge <paquete>
— Elimina paquete y configuración.

snap find <paquete>
— Busca paquetes Snap.
sudo snap install <nombre>
— Instala un Snap.
sudo snap remove <nombre>
— Elimina un Snap.
sudo snap refresh
— Actualiza Snaps instalados.
snap list
— Lista Snaps instalados.
snap info <nombre>
— Información de un Snap.

w
— Usuarios conectados.
sudo adduser <usuario>
— Crea usuario.
sudo deluser <usuario>
— Elimina usuario.
sudo passwd <usuario>
— Establece/cambia contraseña.
su <usuario>
— Cambia de usuario.
sudo passwd -l <usuario>
— Bloquea cuenta.
sudo passwd -u <usuario>
— Desbloquea contraseña.
sudo chage <usuario>
— Configura expiración de contraseña.

id [usuario]
— IDs de usuario y grupo.
groups [usuario]
— Grupos del usuario.
sudo addgroup <grupo>
— Crea grupo.
sudo delgroup <grupo>
— Elimina grupo.

getent passwd [usuario]
— Entrada(s) de usuario desde /etc/nsswitch.conf (local/LDAP, etc.).
getent group [grupo]
— Entrada(s) de grupo.
getent hosts <nombre|IP>
— Resuelve nombres/hosts según NSS.
getent shadow [usuario]
— Muestra hash de contraseña (requiere root).
getent services <nombre|puerto>
— Servicios y puertos (de /etc/services o NSS).
getent passwd | cut -d: -f1
— Lista todos los usuarios definidos.

ip addr show
— Interfaces e IPs.
ip -s link
— Estadísticas de red.
ss -l
— Sockets en escucha.
ping <host>
— Hacer ping a un host.

cat /etc/netplan/*.yaml
— Ver configuración actual.
sudo netplan try
— Probar configuración temporalmente.
sudo netplan apply
— Aplicar configuración.

sudo ufw status
— Estado del firewall.
sudo ufw enable
— Activar UFW.
sudo ufw disable
— Desactivar UFW.
sudo ufw allow <puerto/servicio>
— Permitir tráfico.
sudo ufw deny <puerto/servicio>
— Denegar tráfico.
sudo ufw delete allow|deny <puerto/servicio>
— Borrar regla.

ssh <usuario@host>
— Conexión SSH.
scp <origen> <usuario@host>:<destino>
— Copia segura entre hosts.

LXD ofrece una experiencia unificada para ejecutar y gestionar contenedores o máquinas virtuales.

lxd init
— Inicializa LXD antes del primer uso.

lxc init ubuntu:24.04 <nombre>
— Crea contenedor (no lo inicia).
lxc launch ubuntu:24.04 <nombre>
— Crea e inicia contenedor.
lxc launch ubuntu:24.04 <nombre-vm> --vm
— Crea e inicia VM.

lxc list
— Lista instancias.
lxc info <instancia>
— Información de estado.
lxc start <instancia>
— Inicia.
lxc stop <instancia> [--force]
— Detiene.
lxc delete <instancia> [--force|--interactive]
— Elimina.

lxc exec <instancia> -- <comando>
— Ejecuta dentro de la instancia.
lxc exec <instancia> -- bash
— Shell interactiva (si hay bash).
lxc console <instancia> [flags]
— Consola de la instancia.
lxc file pull <instancia>/<ruta_instancia> <ruta_local>
— Trae archivo.
lxc file push <ruta_local> <instancia>/<ruta_instancia>
— Sube archivo.

lxc project create <proyecto> [--config <opción>]
— Crea proyecto.
lxc project set <proyecto> <opción>
— Configura proyecto.
lxc project switch <proyecto>
— Cambia al proyecto.

Ubuntu Pro amplía el soporte LTS hasta 12 años y añade herramientas de gestión y cumplimiento.

sudo pro attach <token>
— Vincula la máquina usando el token de suscripción.

sudo pro status
— Estado de servicios Pro.
sudo pro enable <servicio>
— Activa un servicio (ESM, FIPS, Livepatch, ...).
sudo pro disable <servicio>
— Desactiva un servicio.

sudo pro enable esm-infra
— Seguridad extendida para infra.
sudo pro enable esm-apps
— Seguridad extendida para apps.

sudo pro enable livepatch
— Aplica parches críticos de kernel sin reiniciar.

sudo pro enable fips
— Activa modo FIPS con estándares criptográficos estrictos.

sudo pro refresh
— Refresca el estado de Ubuntu Pro.
sudo pro detach
— Desvincula la máquina (deshabilita servicios).

#!/bin/bash
— Shebang para indicar el intérprete.
#!/usr/bin/env bash
— Shebang alternativo usando env.
# comentario
— Comentario de una línea.
: <<'EOF' ... EOF
— Comentario multilínea (truco con here-doc).
exit
— Termina el script con el código actual.

var=valor
— Asigna variable (sin espacios).
$var
— Expande el valor de la variable.
readonly var=valor
— Variable de solo lectura.
var=$(comando)
— Sustitución de comandos.
`comando`
— Versión antigua de sustitución.
$0
— Nombre del script; $1, $2... argumentos.
$#
— Número de argumentos.
$?
— Código de salida del último comando.
$$
— PID del script; $! — PID del último proceso en bg.
$HOME, $PATH, $USER
— Variables de entorno comunes.

read -p "Texto: " var
— Pide entrada al usuario.
cmd < input.txt
— Redirige entrada desde archivo.
cmd > out.txt
— Redirige salida (sobrescribe).
cmd >> out.txt
— Redirige salida (añade).
cmd 2> err.txt
— Redirige errores.
cmd &> all.txt
— Junta stdout+stderr.
cmd1 | cmd2
— Pipe: salida de uno, entrada de otro.

if [ condición ]; then ... fi
— If básico.
if ...; then ...; else ...; fi
— If/else.
if ...; then ...; elif ...; else ...; fi
— If/elif/else.
case var in patrón) ... ;; *) ... ;; esac
— Condiciones por patrones.
test condición
— Devuelve 0/1 según condición.

for x in lista; do ...; done
— Itera una lista.
while condición; do ...; done
— Mientras se cumpla.
until condición; do ...; done
— Hasta que se cumpla.
select op in lista; do ...; done
— Menú simple.
break
— Rompe el bucle; continue — Salta a la siguiente iteración.

declare -a arr=(a b c)
— Array indexado; ${arr[0]} primer elemento.
declare -A map; map[clave]=valor
— Array asociativo.
function f() { local x=1; echo $x; }
— Función con variable local.
return n
— Devuelve código de salida de función.

(( a + b )), (( a - b )), (( a * b )), (( a / b )), (( a % b )), (( a ** b ))
— Aritmética.
((i++)) / ((i--))
— Incremento / decremento.
-lt, -gt, -le, -ge, -eq, -ne
— Comparadores numéricos.
[[ -n str ]]
— Cadena no vacía; [[ -z str ]] — Cadena vacía.
&&, ||, !
— AND, OR, NOT lógicos.

echo "${a} ${b}"
— Concatenación simple.
${str:n}
— Subcadena desde índice n hasta el final.
${str:0:5}
— Subcadena (pos 0, longitud 5).
${#str}
— Longitud de la cadena.
[[ $str == *"World"* ]]
— Coincidencia por patrón.
${str/World/Universe}
— Sustituye primera ocurrencia.
${str#\ *}
— Quita prefijo; ${str%%*( )} — Quita sufijos de espacios (extglob).
