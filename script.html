Scripts en Linux

Un script en Linux es un archivo de texto plano que contiene comandos de shell. Los scripts pueden automatizar tareas repetitivas, realizar backups, gestionar permisos de archivos, entre otras tareas.

Cómo crear un script

  1. Crea un archivo de texto plano con la extensión .sh. Por ejemplo, miscript.sh.
  2. Agrega los comandos de shell que quieres ejecutar en el archivo.
  3. Asigna permisos de ejecución al archivo con el comando chmod +x miscript.sh.
  4. Ejecuta el script con el comando ./miscript.sh.

¡Listo! Ahora ya tienes un script funcional en Linux.

Ejemplos Scripts de Linux

Mostrar en pantalla el nombre de usuario actual y la fecha y hora actual.

Solución:

#!/bin/bash
echo "Usuario actual: $(whoami)"
echo "Fecha y hora actual: $(date)"

Mostrar en pantalla una lista de archivos y directorios en el directorio actual.

Solución:

                #!/bin/bash
                ls -a

Contar el número de archivos y directorios en el directorio actual y mostrarlo en pantalla.

Solución:

#!/bin/bash
        echo "Número de archivos: $(ls -1 | wc -l)"
        echo "Número de directorios: $(ls -l | grep -c ^d)"

Crear tres directorios con nombres 'uno', 'dos' y 'tres' en el directorio actual.

Solución:

#!/bin/bash
             mkdir uno dos tres

Copiar todos los archivos con extensión .txt en el directorio actual al directorio /tmp.

Solución:

#!/bin/bash
            cp *.txt /tmp

Este script recibe como parámetro el nombre del usuario y muestra un mensaje de bienvenida.

Solución:

#!/bin/bash
              echo "Bienvenido, $1!"

Para utilizar este script, guarda el código anterior en un archivo con extensión .sh, por ejemplo, "saludar.sh". Luego, asigna permisos de ejecución al archivo con el comando chmod +x saludar.sh. Para ejecutar el script, escribe en la terminal ./saludar.sh nombre_de_usuario, reemplazando "nombre_de_usuario" por el nombre del usuario al que quieres saludar.
